Abstract

Cancer diagnosis is a setting where misclassification errors are highly asymmetric: a false negative can delay treatment, while a false positive can lead to unnecessary biopsies and patient distress. In this project, we study whether full-data Bayesian Gaussian Process Classification can be made practical for histopathology-scale datasets. Gaussian Process Classification provides calibrated probabilistic predictions, but standard inference requires dense kernel operations that are infeasible at PCam scale. We combine accelerated Randomly Pivoted Cholesky, Hamiltonian Monte Carlo in low-rank non-centered coordinates, and a pivots-as-inducing-points prediction rule. This makes MCMC-based GPC tractable on the full PCam training set and improves calibration over SVGP and neural network baselines while remaining competitive on accuracy.

Motivation

Standard neural network classifiers can achieve strong accuracy on histopathology benchmarks, but their softmax probabilities are often overconfident and do not necessarily represent reliable uncertainty. In medical triage, this matters: pathologists need probabilities that can support decisions about which tissue patches require human review.

Gaussian Process Classification is a natural alternative because it gives a posterior over latent functions and produces predictive distributions rather than only point estimates. The computational problem is the kernel matrix: for N = 262,144 PCam training examples, the dense kernel matrix would require roughly 550 GB in float64, and exact Cholesky factorization is not practical.

Problem Setup

Dataset

PCam, a binary histopathology dataset for detecting metastatic tissue in image patches.

Training Scale

N = 262,144 training points and 32,768 held-out test points.

Features

Phikon pathology transformer embeddings, projected to d = 512 dimensions using PCA.

Method

1. Accelerated RPCholesky

We approximate the full kernel matrix with a low-rank factor produced by accelerated Randomly Pivoted Cholesky. Instead of materializing the full N × N matrix, the algorithm queries only selected kernel columns.

F Fᵀ ≈ K,     F ∈ RN×k,     k ≪ N

At PCam scale, we use k = 200, reducing the kernel-evaluation cost from O(N²) to O(Nk).

2. HMC in Non-Centered Coordinates

We reparameterize the GP latent function as:

f = Fν,     ν ∼ N(0, Ik)

This changes the sampling problem from an N-dimensional posterior over latent function values to a k-dimensional posterior over ν. We then run Hamiltonian Monte Carlo with 25 leapfrog steps and dual-averaging step-size adaptation.

3. Pivots-as-Inducing-Points Prediction

The RPCholesky pivot set is reused at test time as inducing points. This avoids evaluating the test kernel against all training points, replacing the naive predictive cost with a much smaller pivot-based computation.

Computational Cost

Kernel Factorization

16 sec

Accelerated RPCholesky on the full PCam training set.

HMC Sampling

~20 min

1000 post-warmup samples with 25 leapfrog steps.

Prediction

~2 sec

Pivot-based predictive inference on the test set.

The full pipeline runs in approximately 22 minutes on a single GPU. Dense Cholesky at this scale would require cubic time and hundreds of gigabytes of memory, making the direct approach infeasible.

Experimental Progression

Experiment 1: Small-Scale Validation

We first tested the method on N = 2000 synthetic blob data and PCam subsamples. These experiments established that accelerated RPCholesky reduces runtime and that HMC mixes better than Random-Walk Metropolis and MALA on the PCam posterior.

Experiment 2: Full-Scale PCam Benchmark

We scaled to the full PCam training set using N = 262,144, d = 512 Phikon embeddings, and RPCholesky rank k = 200. We compare against SVGP and a Residual MLP trained on the same embeddings.

Experiment 3: Calibration and Posterior Uncertainty

We analyze whether the low-rank HMC GP improves probabilistic metrics such as ECE and NLL, where calibrated uncertainty matters more than raw classification accuracy alone.

Final Results

Method Accuracy AUROC NLL ECE Brier
GPyTorch SVGP 0.882 0.971 0.087 0.101 0.087
Residual MLP 0.877 0.965 0.111 0.116 0.111
Low-rank HMC GP, ours 0.914 0.978 0.064 0.055 0.064

Our method achieves the highest accuracy and AUROC while substantially improving calibration. The ECE reduction is especially important in a diagnostic setting, where confidence scores may be used to decide which patches should be reviewed by a human pathologist.

Clinical Interpretation

Accuracy and AUROC measure discrimination, but they do not fully capture whether a model's probabilities are usable for decision-making. Expected Calibration Error measures the gap between stated confidence and empirical correctness. In cancer screening, this directly affects whether confidence thresholds are meaningful.

Our model reduces ECE from 0.101 for SVGP to 0.055, and from 0.116 for the MLP to 0.055. This indicates that the HMC posterior better represents uncertainty than the variational approximation used by SVGP or the point-estimate predictions used by the MLP.

Additional Findings

Lower Rank Can Improve Generalization

A rank sweep over k ∈ {100, 200, 300, 500} showed that predictive performance peaked at k = 200, even though higher ranks gave tighter kernel approximations. This suggests that the low-rank approximation also acts as implicit regularization.

HMC Mixes Efficiently

In the low-rank posterior, longer HMC trajectories substantially reduce autocorrelation time. With 25 leapfrog steps, HMC reaches efficient mixing while remaining computationally feasible.

Conclusion

This project demonstrates that MCMC-based Gaussian Process Classification is now tractable on histopathology-scale datasets when combined with accelerated RPCholesky and low-rank HMC inference. The method is not as fast as SVGP or neural network baselines, but it makes calibrated Bayesian classification deployable at a scale where full GPC was previously infeasible.

The main result is not only improved accuracy, but improved uncertainty quantification. For domains such as medical diagnosis, where confidence scores affect downstream human review, this calibration improvement is central to the value of the method.

Future Work

  • Extend the method to multiclass medical classification tasks.
  • Run stronger hyperparameter sweeps for neural network baselines.
  • Reconcile the HMC sigmoid likelihood with the SVGP probit likelihood for cleaner comparisons.
  • Explore whether the observed low-rank regularization effect generalizes beyond PCam.
  • Test hybrid architectures that combine learned image features with principled Bayesian probabilistic heads.
